### 班级社交系统
## 技术栈
- 后端：SpringBoot + Mybatis-Plus + Sa-token + Redis
- 前端：Vue3 + Element-Plus + Pinia + Vue-Router4

## 注意事项
- 项目启动之前先安装并启动redis。
- 启动前端：
- npm install #安装依赖
- npm run dev #启动
